What is Autograph?

Autograph is a load testing tool that spits out easy to understand HTML reports. Autograph aims to be the easiest way to get a peek at how your app responds to load.

Features

  • Bulit on the powerful httperf tool
  • Generates sexy reports
  • Simple to use

Limitations

Autograph's testing strategy is rather basic: slam the specified URLs with GETs, and see how they respond. Here are a few things autograph does NOT do:

  • Handle POST, HEAD, PUT, DELETE requests
  • Test with specified user-scenarios
  • Apply load from multiple servers
